How to Paste Into GitHub

  1. Go to github.com/JarradHenderson/awards-desk
  2. Open index.html and click the pencil (edit) icon
  3. Find the CONFIG = { line (around line 140)
  4. Replace the entire CONFIG block with the code above
  5. Scroll down, add a commit message (e.g., "Update competitions and deadlines")
  6. Click "Commit changes"
  7. Wait 1-2 minutes for GitHub Pages to rebuild
  8. Hard-refresh the form (Cmd+Shift+R) to see updates
